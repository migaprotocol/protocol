## Overview

MIGA Protocol is the token launch site for MIGA - a Freedom of Information DAO funding anti-censorship technology and open internet access for Iran.

**Key Features:**
- Fair launch on Solana (no presale, no VCs, no team allocation)
- 1 MIGA = 1 Vote in community governance
- 100% of funds go to community treasury

## Development

```bash
# Install dependencies

# Run development server
pnpm dev

# Build for production
pnpm build
```
